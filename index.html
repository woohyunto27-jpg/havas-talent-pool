📖 如何获取 JSONBin API Key

  1. 访问 jsonbin.io 并注册账号（支持 Google 登录，免费）
  2. 注册后进入 API Keys 页面
  3. 点击「Create API Key」，复制生成的 Key
  4. 将 Key 粘贴到上方输入框，点击「连接云同步」
  5. 系统会自动创建数据仓库并同步，无需其他配置 ✅
💡 为什么选择 JSONBin？ Firebase 使用 Google 服务器，国内无法访问。JSONBin 是轻量级 JSON 存储服务，国内直连无障碍，免费额度每日 1 万次请求，完全够用。
